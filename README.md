# board

**1. 개요**
- 프로젝트 이름 : Simple Board Service
- 개발 인원 : 전용본(1명)
- 개발 기간 : 2023-05-26 ~ 
  - 요구사항 분석과 빌드 세팅을 해놓은 후에 인프런 강의 및 기말 고사가 끝나는 직후 개발 시작 
- 주요 기능
  - CRUD
  - 회원가입, 로그인
- 개발 언어 : Java 11
- 개발 환경 : SpringBoot 2.7.12, JPA, Thymeleaf
- 데이터베이스 : MySQL
